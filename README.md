# RACMO IVT pipeline (Arctic)

Processing scripts to compute Integrated Vapor Transport (IVT)
from RACMO output for Arctic analysis.

## Contents
- *.sh : shell scripts for preprocessing and CDO operations
- *.py : Python scripts for diagnostics and checks

## Notes
- Large data files are excluded via .gitignore
- Pipeline developed at UU / IMAU
